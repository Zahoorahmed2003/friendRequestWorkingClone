# Simple Friend Management Webpage

This project is a simple webpage built with HTML, CSS, and JavaScript that simulates the "Add Friend" and "Unfriend" functionality similar to Facebook.

## Features

- **Add Friend Button**: Adds a friend when clicked.
- **Unfriend Button**: Removes a friend when clicked.
- **Dynamic Updates**: The buttons update dynamically based on the current state (i.e., if the user is a friend or not).
